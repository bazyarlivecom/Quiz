# راهنمای همگام‌سازی با Branch b2

این پروژه به صورت پیش‌فرض با branch `b2` همگام‌سازی می‌شود.

## دستورات همگام‌سازی

### دریافت آخرین تغییرات از b2:
```bash
git pull
```
یا به صورت صریح:
```bash
git pull origin b2
```

### ارسال تغییرات به b2:
```bash
git push
```
یا به صورت صریح:
```bash
git push origin b2
```

### بررسی وضعیت:
```bash
git status
```

### مشاهده تفاوت با remote:
```bash
git fetch origin
git log HEAD..origin/b2
```

## تنظیمات فعلی

- **Branch فعلی**: `b2`
- **Remote tracking**: `origin/b2`
- **Push default**: `simple` (فقط به branch tracking شده)
- **Pull strategy**: `merge` (نه rebase)

## نکات مهم

1. همیشه قبل از push، `git pull` کنید تا مطمئن شوید آخرین تغییرات را دارید
2. اگر conflict پیش آمد، ابتدا آن را حل کنید سپس commit کنید
3. برای تغییر branch، از `git checkout` استفاده کنید
